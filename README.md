# Smart Parking System

A desktop application designed to reduce parking congestion in malls and large facilities by allowing users to reserve parking spaces before arrival.

## Features

- User Authentication
- Driver Dashboard
- Administrator Dashboard
- Parking Reservation
- Automatic Slot Release
- Real-Time Countdown Timer
- Role-Based Access Control

## Technologies

- C#
- Windows Forms
- SQLite
- .NET Framework
- UML
